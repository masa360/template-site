# LPテンプレート作成サイト

ジャンル（物販・美容・スクール・体験）を選び、スタイルを選んでLPを作成し、LINE公式アカウントと紐づけるまでを一通り行えるホームページです。

## 機能

1. **ジャンル選択** … 物販 / 美容 / スクール / 体験もの から1つ選択
2. **スタイル選択** … 共通の「スタンダード（モダン）」デザインで作成
3. **内容入力** … メイン画像URL・キャッチコピー・ヒアリング内容・会社概要・特徴を入力
4. **LINE紐づけ** … LINE公式アカウントのID（@xxx）または電話番号、友だち追加URLを入力して紐づけ
5. **完了** … 作成データはブラウザのローカルストレージに保存。LPプレビューリンクで確認可能

## 使い方（ローカル）

1. このフォルダをそのまま開き、`index.html` をブラウザで開くだけでは **プレビュー用のクエリ付きURLが正しく動かない** ことがあります（file プロトコル制限のため）。
2. **ローカルサーバーで表示することを推奨** します。

### 方法A: VS Code / Cursor の Live Server

- 拡張機能「Live Server」を入れ、`index.html` を右クリック → 「Open with Live Server」

### 方法B: コマンドで簡易サーバーを起動

```bash
# Python が入っている場合（Python 3）
cd "d:\cursur\LP\テンプレートサイト"
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開く。

```bash
# Node が入っている場合（npx で one-time サーバー）
cd "d:\cursur\LP\テンプレートサイト"
npx -y serve -p 8080
```

ブラウザで `http://localhost:8080` を開く。

## デプロイ時の注意

- 現状は **クライアント側のみ** で完結しています。LINEの「番号入力で紐づけ」は、入力内容を **ローカルストレージ** に保存しています。
- デプロイ後に「サーバー側でLINEと本当に紐づける」場合は、バックエンドAPIを用意し、`js/app.js` の LINE フォーム送信時にそのAPIへ送信するように変更してください。
- プレビューは、入力内容をクエリパラメータで各テンプレート（`templates/*.html`）に渡しています。デプロイ後も同じ構成で動作します。

## カスタマイズ

- **画像** … 入力フォームの「メイン画像」に表示したい画像のURLを入力してください（後から差し替え可能）。
- **テンプレートの文言** … `templates/` 内の各HTMLの初期テキストを編集すると、未入力時の表示を変更できます。
